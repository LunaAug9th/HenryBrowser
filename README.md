# HenryBrowser
### lightweight browser with TypeScript and Electron

## How to Build and Run


1. Clone The Repository
```bash
mkdir HenryBrowser
cd HenryBrowser
git clone https://github.com/LunaAug9th/HenryBrowser
```

2. Change Dir
```bash
cd HenryBrowser
```

3. Configure Config.json and set Data Dir
<p style="margin-left: 25px; font-size: 14px; font-weight:450;"> Default is stored in the OS's default app data path and can be changed to absolute/relative path </p>

4. Install the Dependences
```bash
npm install
```

5. Build
```bash
npm run build
```

6. Run
```bash
npm start
```
